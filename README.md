# Calendly Clone - Scheduling Platform

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MySQL / PostgreSQL

## Features
- Create event types
- Set availability
- Public booking page
- Prevent double booking
- Booking confirmation
- Email notifications (bonus)

## Setup Instructions

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev

## Assumptions
- No login required (as per assignment)
- Single default user
