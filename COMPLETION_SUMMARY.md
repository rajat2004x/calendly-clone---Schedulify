## 🚀 How to Run

### Start Backend
```bash
cd backend
node server.js
```
➜ Running on `http://localhost:5000`

### Start Frontend
```bash
cd frontend
npm run dev
```
➜ Running on `http://localhost:5174`

### Visit Application
Open `http://localhost:5174` in your browser
